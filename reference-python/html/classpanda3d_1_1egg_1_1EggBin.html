EggBin Class Reference

A type of group node that holds related subnodes. This is a special kind of node that will never be read in from an egg file, but can only exist in the egg scene graph if it is created via the use of an EggBinMaker. More...

Detailed Description

A type of group node that holds related subnodes. This is a special kind of node that will never be read in from an egg file, but can only exist in the egg scene graph if it is created via the use of an EggBinMaker.
